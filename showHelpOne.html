User Access Portal Setup

User Access Portal Setup

User Access Portal Enable: Enabling this will Block traffic coming from LAN Network.User Access Portal Profile: Profiles can be used for User Access Portal Login page.Redirection Type: Allows the user to configure traffic redirection for http or https traffic.Save: Click to save the settings.Cancel: Click to revert to the previous settings.

List of Available Profiles

Any one of these profiles can be used for User Access Portal Login page while enabling User Access Portal.Profile Name: Name of the profiles that are available.Status: Lets user know if the profile is in use or not.The actions that can be performed on profiles are:Select All: Selects all the profiles in the table.Edit: The Edit button will link to the User Access Portal Profile Configuration page, allowing you to make changes to the selected profile. The Default Profile cannot be edited.Delete: Deletes the selected profile. The default profile and the current profile being used, cannot be deleted.Add: Clicking this button will link to the User Access Portal Profile Configuration page. Maximum allowed number of profiles is 5, excluding the default one.Preview: Shows preview of the page, if a profile is selected.

User Access Portal Configuration

User Access Portal login page display can be altered by modifying the configuration.

General Details

Profile Name: Name of the profile that is being added.Browser Title: It is the browser title.Page Background Color: Choose the background color for the page from list. Choose 'Custom' to mention a custom color.Custom Color: If page background color selected as 'Custom', mention the custom color in hexadecimal notation (HEX) for the combination of Red, Green, and Blue color values (RGB) like #FFFFFF.

Header Details

It allows user to configure how the header portion of the page should be displayed.Header Logo Image: Sets the logo for the header portion.Add: Will let you add a new image. This image can be set as header image for this profile.Header Caption: Text to be displayed in the header portion.Caption Font: Font for the header text to be displayed.Font Size: Font size for the header text to be displayed.Font Color: Color in which the text is to be displayed.

Login Details

It allows user to configure how the Login Area of the page should be displayed.Login Section Title: Title for the Login BoxWelcome Message: Message which is displayed when a user visits the page.Error Message: Error Message displayed when user enters invalid credentials.

Advertisement Details

It allows user to configure advertisements on the page.Enable Advertisement: Choose this to enable advertising on the page. Ad Place: Choose the position (left / right) to place the Advertisement. Ad Content: Write advertisement information here. Font: Font for the information to be displayed.Font Size: Font size for the information to be displayed.Font Color: Color in which the information is to be displayed.

Footer Details

It allows user to configure the footer portion of the page.Change Footer Content: Choose this to enable footer portion in the page. Footer Content: Write the footer content here. Footer Font Color: Color in which the footer is to be displayed.Click Save to save your changes.Click Cancel to revert to the previous settings.

Following drop down/buttons are available for assisting in pagination

Show 10/25/50/100 entries Allows the user to select to display 10/25/50/100 entries in one page.First Navigates to the first page of the displayed entries.Previous Navigates to the previous page of the displayed entries.Next Navigates to the next page of the displayed entries.Last Navigates to the last page of the displayed entries.
